# Notice

NTE HDR Panel is an unofficial local configuration helper for Neverness To Everness / Ananta.

It is not affiliated with Hotta Studio, Perfect World Games, NVIDIA, Microsoft, or Epic Games.

The tool writes Unreal Engine renderer settings to a local user configuration file and provides backup / restore automation. It does not include third-party DLL assets.

## Reference Project

This project references the structure and product workflow of:

- https://github.com/llg1634/nte-dlss-panel

Referenced ideas include local WebUI operation, manifest-based backup and restore, HUD verification, backend shutdown from the UI, and GitHub-ready documentation layout.

No DLSSTweaks binary asset is included in this HDR project.
