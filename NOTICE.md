# Notice

NTE HDR Panel is an unofficial local configuration helper for Neverness To Everness / Ananta.

It is not affiliated with Hotta Studio, Perfect World Games, NVIDIA, Microsoft, or Epic Games.

The tool writes Unreal Engine renderer settings to a local user configuration file and provides backup / restore automation. It does not include third-party DLL assets.

## Related Community Projects

This HDR panel does not bundle or depend on the following projects. They are credited and referenced because they are important community tools in the DLSS / HDR / PC graphics tweaking ecosystem, and because this project explicitly differs from them:

- DLSSTweaks by emoose: https://github.com/emoose/DLSSTweaks and https://www.nexusmods.com/site/mods/550
- RenoDX by ShortFuse and contributors: https://github.com/clshortfuse/renodx
- ReShade by crosire and contributors: https://github.com/crosire/reshade and https://reshade.me/
- Special K by Kaldaien and contributors: https://github.com/SpecialKO/SpecialK and https://www.special-k.info/

No DLSSTweaks, RenoDX, ReShade, or Special K binary is included in this repository. This project does not inject into the game process and does not modify the game installation directory.

## Reference Project

This project references the structure and product workflow of:

- https://github.com/llg1634/nte-dlss-panel

Referenced ideas include local WebUI operation, manifest-based backup and restore, HUD verification, backend shutdown from the UI, and GitHub-ready documentation layout.

No DLSSTweaks binary asset is included in this HDR project.
