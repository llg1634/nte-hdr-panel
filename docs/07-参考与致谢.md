# 参考与致谢

## 相关社区项目

这些项目不是本工具依赖，也没有被打包进本仓库；列在这里是为了说明本工具和它们的边界。

## DLSSTweaks

- GitHub：https://github.com/emoose/DLSSTweaks
- NexusMods：https://www.nexusmods.com/site/mods/550
- 作者：emoose

DLSSTweaks 是 DLSS wrapper / tweak 工具，可调整 DLSS scaling ratios、DLSS presets、DLAA 等。  
本项目不包含 DLSSTweaks DLL，不写 `dlsstweaks.ini`，不放 `winmm.dll` / `dxgi.dll` / `nvngx.dll`。

## RenoDX

- GitHub：https://github.com/clshortfuse/renodx
- Mods Wiki：https://github.com/clshortfuse/renodx/wiki/Mods

RenoDX 是 DirectX 游戏画面改造工具集，常用于 HDR / tonemapping / shader 改造。它基于 ReShade add-on 系统。  
本项目不是 RenoDX add-on，不改 shader，不改 swapchain，不做 tonemapping retrofit。

## ReShade

- GitHub：https://github.com/crosire/reshade
- 官网：https://reshade.me/

ReShade 是通用 post-processing injector 和 shader/effect 平台。  
本项目不是 ReShade preset，不加载 ReShade，不做后处理滤镜。

## Special K

- GitHub：https://github.com/SpecialKO/SpecialK
- 官网：https://www.special-k.info/

Special K 是 PC 游戏增强和诊断工具，包含性能分析、注入、HDR 改造等能力。  
本项目不是 Special K 插件，不使用 local/global injection。

## 本项目的不同点

`nte-dlss-panel` 处理 DLSS / DLSSTweaks 低渲染倍率。

`nte-hdr-panel` 只处理 UE 原生 HDR 配置：

```text
%LOCALAPPDATA%\HT\Saved\Config\Windows\Engine.ini
```

本项目不复制 DLL，不修改游戏安装目录，不读取游戏模块。
