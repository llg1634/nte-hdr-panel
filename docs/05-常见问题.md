# 常见问题

## 这是不是 RTX HDR？

不是。这个工具写入 UE RendererSettings，让游戏走 UE HDR 输出配置。

## 这是不是 DLSSTweaks？

不是。DLSSTweaks 是 DLSS wrapper / tweak 工具。本项目不包含 DLSSTweaks DLL，也不会写 `dlsstweaks.ini`。

## 这是不是 RenoDX / ReShade / Special K？

不是。本项目不是 RenoDX add-on，不加载 ReShade，不使用 Special K injection。它只写 `Engine.ini`。

## 为什么要只读？

本机测试发现，不设只读时游戏启动阶段会重写 `Engine.ini`。设为只读后配置能保留。

## 只读会不会影响恢复？

不会。恢复前工具会先取消当前文件只读，再按 manifest 恢复原文件和原只读状态。

## 我想直接回到最初未配置 HDR 的版本怎么办？

用面板里的：

```text
恢复未配置 HDR 的原版
```

这个按钮不会删文件重建，也不会让游戏下次自己生成；它只会恢复当前找到的最早那份未配置 HDR 的 `Engine.ini` 备份。

## HUD 开关失败怎么办？

用管理员权限启动：

```text
run_as_admin.bat
```

## 游戏提示黑客工具怎么办？

先关闭这些工具：

```text
Procmon64 / Process Monitor
Process Explorer
x64dbg / x32dbg
Cheat Engine
IDA
Frida
```

本工具不会读取游戏模块，也不会注入游戏进程。

## 怎么确认成功？

优先看 NVIDIA HUD：

```text
DX12 / D3D12
HDR Mode: Custom
```

同时确认 `Engine.ini` 没被游戏重写。
