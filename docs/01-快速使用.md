# 快速使用

## 推荐流程

1. 关闭异环客户端和启动器。
2. 运行 `NTEHDRPanel.exe` 或双击 `run.bat`。
3. 打开 `http://127.0.0.1:22533`。
4. 点击“检测配置”。
5. HDR 参数先使用 `1000 / 18 / 1.0`。
6. 保持“写入后设为只读”勾选。
7. 点击“备份并写入原生 HDR”。
8. 启动异环。
9. 用 NVIDIA HUD 看是否显示 `DX12` / `D3D12` 和 `HDR Mode: Custom`。

## 如果目录不存在

配置目录通常是：

```text
%LOCALAPPDATA%\HT\Saved\Config\Windows
```

如果目录不存在，先启动一次游戏并进入主界面或大世界，让游戏生成配置目录。

## 如果画面偏暗

优先提高：

```ini
r.HDR.Display.MidLuminance
```

可以试：

```text
120
250
```

显示器峰值亮度对应：

```ini
r.HDR.Display.MaxLuminance
```
