# 原理与测试结论

## 原理

异环是 UE 游戏，UE 支持通过 RendererSettings 开启 HDR 输出：

```ini
[/Script/Engine.RendererSettings]
r.HDR.EnableHDROutput=1
r.HDR.UI.Level=1.0
r.HDR.Display.MaxLuminance=1000
r.HDR.Display.MidLuminance=18
```

这个方法走 UE HDR 输出管线，不是 RTX HDR，也不是 Windows 自动 HDR。

## 本机测试结论

测试机上观察到：

1. 游戏原始 `Engine.ini` 是 200 字节的编码内容。
2. 写入明文 HDR 配置后直接启动游戏，文件会被删除或重写。
3. 写入后设为只读，配置能保留。
4. NVIDIA HUD 可显示 `DX12` 和 `HDR Mode: Custom`。

因此工具默认写入后设为只读。

## 检测策略

工具只做普通检测：

```text
读取 Engine.ini
读取普通进程名
读取 NVIDIA HUD 注册表
```

工具不做：

```text
读取游戏模块
注入游戏进程
使用 Process Monitor
修改游戏安装目录
```

这样可以降低触发“黑客工具”检测的概率。
